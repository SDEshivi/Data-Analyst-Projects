# Pizza Sales Dashboard

This project analyzes pizza sales data to uncover meaningful business insights and trends using Excel and Power BI.

## Tools & Technologies

* Microsoft Excel
* Power BI

## Key Performance Indicators (KPIs)

* Total Revenue
* Total Orders
* Total Pizzas Sold
* Average Order Value
* Average Pizzas per Order

## Dashboard Insights

* Sales distribution by pizza category
* Sales performance by pizza size
* Order trends by weekday
* Top 5 and bottom 5 selling pizzas

## Project Objective

The goal of this project is to analyze pizza sales data and build an interactive dashboard that helps businesses understand sales performance, customer ordering patterns, and product popularity.
